# 快捷键指南

## 全局快捷键

| 功能 | macOS | 说明 |
|------|-------|------|
| 显示/隐藏窗口 | `Cmd + Alt + \`` | 快速唤起或隐藏主窗口 |
| 快速粘贴 | `Cmd + Ctrl + V` | 粘贴最近一项内容 |
| 显示历史 | `Cmd + F9` | 打开历史记录界面 |
| 搜索 | `Cmd + Shift + F` | 打开搜索栏 |
| OCR | `Cmd + F8` | 图片文字识别 |

## 应用内快捷键

| 功能 | 快捷键 |
|------|--------|
| 隐藏窗口 | `Cmd + H` |
| 退出应用 | `Cmd + Q` |
| 关闭窗口 | `Cmd + W` |
| 打开设置 | `Cmd + ,` |

## 自定义快捷键

在 `设置` → `常规` → `全局快捷键` 中修改。

## 禁止注册的快捷键

以下系统快捷键不可覆盖：

- `Cmd + C/V/X` (复制/粘贴/剪切)
- `Cmd + Z/A` (撤销/全选)
- `Cmd + S/Q` (保存/退出)
- `Cmd + Tab` (切换应用)

## 故障排查

### 快捷键不响应

1. 检查是否有其他应用占用相同快捷键
2. 确认应用有辅助功能权限（macOS）
3. 重启应用

### 应用最小化后无法唤起

1. 确认应用仍在运行（检查 Dock 栏）
2. macOS 需要在"隐私与安全性"中开启辅助功能权限

### 后台快捷键失效

1. 确认 macOS 辅助功能权限已开启
2. 检查 entitlements 配置

### macOS 权限设置

1. 系统设置 → 隐私与安全性 → 辅助功能
2. 添加 ClipFlow 并启用
3. 重启应用
